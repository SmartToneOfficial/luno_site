Пользовательское соглашение

1. Общие положения

1.1. Настоящее Пользовательское соглашение (далее — Соглашение) регулирует отношения между администрацией бота Luno (далее — Администрация) и пользователем (далее — Пользователь) по использованию бота Luno.

2. Права и обязанности сторон

2.1. Администрация обязуется:

2.1.1. Обеспечивать работоспособность бота в соответствии с его функциональным назначением.

2.1.2. Не передавать персональные данные Пользователей третьим лицам.

2.2. Пользователь обязуется:

2.2.1. Не использовать бота для рассылки спама, вредоносного ПО или иных противоправных действий.

2.2.2. Соблюдать правила Discord и законодательство своей страны.

3. Ответственность

3.1. Администрация не несет ответственности за действия Пользователей, совершенные с использованием бота.

3.2. Администрация оставляет за собой право блокировать доступ к боту для Пользователей, нарушающих настоящее Соглашение.

4. Заключительные положения

4.1. Администрация оставляет за собой право вносить изменения в настоящее Соглашение без предварительного уведомления Пользователей.
