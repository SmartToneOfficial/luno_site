Политика конфиденциальности

1. Сбор информации

1.1. Бот Luno может собирать и хранить следующую информацию:

- ID пользователей Discord

- ID серверов Discord

- Настройки серверов, связанные с функционалом бота

2. Использование информации

2.1. Собранная информация используется исключительно для:

- Обеспечения работоспособности функций бота

- Улучшения качества предоставляемых услуг

- Предотвращения злоупотреблений

3. Защита информации

3.1. Администрация принимает все необходимые меры для защиты персональных данных Пользователей от несанкционированного доступа.

4. Раскрытие информации третьим лицам

4.1. Администрация не передает персональные данные Пользователей третьим лицам, за исключением случаев, предусмотренных законодательством.
